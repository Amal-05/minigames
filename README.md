# 🎮 Cartoonish Mini-Games Hub

Welcome to the ultimate fun zone! A collection of 7 highly addictive, beautifully styled, and mobile-responsive mini-games heavily inspired by the quirky, playful style of neal.fun.

### 🕹️ Play Now
[![Play Game](https://img.shields.io/badge/PLAY_GAME-FF5E5E?style=for-the-badge&logo=vercel&logoColor=white)](https://minigames-pied.vercel.app/)

## 👾 Games Included
1. **Snake** - Classic snake action with a mobile-friendly D-Pad!
2. **Tic Tac Toe vs AI** - See if you can beat the smart JS logic AI.
3. **Perfect Circle** - Trace perfectly around the dot; tracks mouse and touch variance!
4. **Reaction Time** - Test your raw reflexes.
5. **Aim Trainer Frenzy** - You have 30 seconds to click as many targets as you can!
6. **Falling Words** - Type technical words before they hit the bottom and kill your combo!
7. **Color Stack Sort** - The famous water-sort style brain teaser puzzle.

## ✨ Features
- 📱 **100% Mobile Responsive** with precise touch-first controls (Pointer events, Touch listeners).
- 🎶 **Web Audio System** - Procedural click tones, eating noises, and an endlessly cycling retro 8-bit chip-tune!
- 🎨 **Neo-Brutalist & Glassmorphic styles** - A bold cartoonish menu transforms into ultra-premium full-screen game interfaces.
- ⚡ **Zero Dependencies** - Built exclusively with raw HTML, CSS, and Vanilla JavaScript.
